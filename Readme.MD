# 📰 GenAI News Digest Pipeline

**GenAI News Digest Pipeline** is a production-ready AI system that automatically collects, analyzes, summarizes, ranks, and delivers the latest AI news every day.  
It converts raw content from YouTube channels, blogs, and RSS feeds into structured, personalized daily email digests — powered entirely by fast, free modern LLMs.

This project demonstrates how real GenAI products are engineered:

**scraping → processing → LLM reasoning → ranking → digest creation → email delivery → deployment**

Clean. Modular. Scalable. Production-ready.

---

## 🚀 What This Project Delivers

- Automated news collection from multiple AI-focused sources  
- Transcript & article extraction  
- Ultra-fast summarization using **Groq Mixtral / LLaMA 3**  
- Digest writing & polishing using **Gemini 2.5 Flash**  
- Text cleaning and normalization using **HuggingFace Transformers**  
- User-profile-based ranking and article relevance scoring  
- Polished HTML email digests delivered automatically  
- Fully automated daily execution pipeline  
- Docker & Render deployment ready  

A complete GenAI workflow from ingestion to delivery.
