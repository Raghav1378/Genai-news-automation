## � Introduction

**NeuralPulse** is a sophisticated News Intelligence System designed to combat information overload. It leverages a multi-model LLM architecture to pull, verify, and distill global news into actionable intelligence. Unlike standard search engines, NeuralPulse interrogates sources for bias, checks factual claims, and cross-references data points using multiple independent AI models.

---

## 🚀 Core Features

### 🔍 Advanced Search & Synthesis
- **Real-time Engine**: Powered by **Tavily AI** for deep, relevant web crawling.
- **Dual-Model Verification**: Every analysis is generated by **Groq (Llama 3.3)** and independently cross-verified by **Google Gemini 2.5 Flash**.
- **Deep Research Mode**: Automatically spawns recursive research tasks when conflicting information is detected.

### �️ Truth & Integrity
- **Automated Fact-Checking**: Extracts and assesses factual claims with evidence-based status (Verified, Unconfirmed, Contradictory).
- **Bias Balancer**: Detects media bias and provides counter-perspectives to ensure a balanced worldview.
- **Source Transparency**: Direct relevance scoring and source attribution for every data point.

### 🧠 Semantic Intelligence
- **RAG Memory**: Integrated **ChromaDB** allows the system to semantically search through its own history of analyses.
- **Trend Analytics**: Dynamic visualization of sentiment and confidence trends across topics using Chart.js.
- **Global Reach**: Search in **11+ major languages** with real-time translation and synthesis.

### 📦 Professional Utilities
- **Pro Export**: Generate high-fidelity **PDF reports** for stakeholders.
- **Voice Anchor**: Anchor-style **spoken briefings** generated for auditory consumption.
- **Pulse Watchlist**: Monitor specific topics and receive automated **HTML email digests**.

## ⚙️ Quick Start

### Local Deployment
```bash
# Clone and Enter
git clone https://github.com/Raghav1378/Genai-news-automation.git
cd Genai-news-automation

# Setup Environment
python -m venv venv
# Windows: venv\Scripts\activate | Unix: source venv/bin/activate
pip install -r backend/requirements.txt

# Run Server
cd backend
python -m uvicorn main:app --reload
```
*Note: Ensure your `TAVILY_API_KEY` is set in the `.env` file.*
